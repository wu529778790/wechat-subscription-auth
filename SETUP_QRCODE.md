# 二维码配置快速指南

## 🎯 你需要做什么

打开 `pages/index.vue` 文件，修改这两行：

```javascript
// 第 143 行：你的公众号名称
const wechatName = ref('你的公众号名称');

// 第 144 行：二维码图片URL（可选）
const qrcodeUrl = ref(''); // 留空显示占位符，或填入图片URL
```

---

## 📸 添加二维码的 3 种方法

### 方法1：使用微信官方二维码（推荐）

1. 登录微信公众号后台
2. 进入 **设置与开发** → **公众号设置**
3. 找到 **公众号二维码**，下载图片
4. 上传到图床（如 GitHub、阿里云OSS等）
5. 复制图片URL，填入：

```javascript
const qrcodeUrl = ref('https://你的图床.com/qrcode.jpg');
```

### 方法2：使用草料二维码（最快）

1. 访问 https://cli.im/
2. 输入你的公众号名称
3. 生成二维码并下载
4. 上传到任意图床
5. 填入URL

### 方法3：使用 Base64（无需图床）

```javascript
// 将图片转换为 Base64（在线工具：base64-image.de）
const qrcodeUrl = ref('data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...');
```

## 💡 小贴士

### 二维码尺寸建议
- 推荐：400x400 px
- 最小：200x200 px
- 格式：PNG（透明背景）

### 如果没有二维码
可以只配置公众号名称：
```javascript
const wechatName = ref('我的公众号');
const qrcodeUrl = ref(''); // 留空
```

用户会看到：
```
微信搜索公众号：我的公众号
```

### 测试配置
修改后，访问你的网站：
```
https://your-domain.com
```

应该能看到：
- 美观的认证弹窗
- 二维码区域（如果配置了）
- 输入框和验证按钮
